Quasardb Java API
=================

Java API for [quasardb](http://www.quasardb.net/) with examples for Hadoop and Mahout.

### Requirements

1. quasardb daemon
2. quasardb JNI API
3. Maven 3
4. JDK 6 or above

### Build instructions:

    mvn clean install
